About Alexa

Alexa is a lightweight, single‑file HTML template for an anime catalog. It includes search, genre filters, and a clean aesthetic inspired by neon Tokyo nights and sakura tones.

  • Semantic HTML, responsive CSS grid, no build step required.
  • Keyboard-friendly: use / to focus search, Esc to clear, and ↑/↓ to jump to top/bottom.
  • Easy to connect to your backend: replace the static card markup with your data loop.
